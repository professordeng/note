---
title: 常用的 DOS 命令
---

这里熟悉一下 DOS 里常用的命令，在 Windows 的 cmd 窗口上可以直接运行。

| Usage            | Command               |
| ---------------- | --------------------- |
| 查看局域网 IP    | `ipconfig`            |
| 局域网远程       | `mstsc`               |
| 关机             | `shutdown -s`         |
| 一小时后自动关机 | `shutdown -s -t 3600` |
| 取消自动关机     | `shutdown -a`         |
